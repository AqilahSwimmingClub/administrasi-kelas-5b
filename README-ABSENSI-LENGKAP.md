# Pembaruan Modul Absensi Lengkap

Fitur yang ditambahkan:
- Absensi harian dengan tanggal otomatis.
- Input dinonaktifkan pada Sabtu, Minggu, dan tanggal libur yang dikonfigurasi.
- Rekap bulanan.
- Rekap Semester 1 (Juli–Desember) dan Semester 2 (Januari–Juni).
- Rekap tahunan sesuai tahun pelajaran.
- Persentase kehadiran per siswa.
- Export Excel dan Cetak/PDF berdasarkan rekap aktif.
- Peringatan kehadiran di bawah 90% dan alpa berulang.
- Riwayat perubahan absensi (audit log).
- Notifikasi otomatis setelah absensi disimpan.
