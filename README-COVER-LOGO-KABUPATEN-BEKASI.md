# Perbaikan Cover Rapor

Logo Kabupaten Bekasi pada cover rapor telah diganti menggunakan file PNG terbaru dari pengguna.
Logo tampil di bawah tulisan SEKOLAH DASAR (SD), berukuran lebih kecil dan proporsional.
Referensi asset menggunakan cache-busting `?v=3` agar browser tidak menampilkan logo lama dari cache.
