# Fitur Informasi Orang Tua

Tambahan ini tidak mengganti modul lama. Data memakai objek sinkronisasi Supabase yang sama.

## Menu Wali Kelas
- Jadwal & Tugas
- Nilai & Hasil Ujian
- Informasi Sekolah dan pembayaran
- Catatan Guru per siswa
- Pesan Langsung privat dari guru
- Galeri foto/video kegiatan

Informasi dapat ditujukan kepada semua orang tua atau satu siswa tertentu.

## Portal Orang Tua
- Melihat absensi harian secara real-time
- Melihat seluruh informasi yang ditujukan kepada anak
- Melihat nilai
- Mengganti foto profil
- Mengunggah Kartu Keluarga
- Mengunggah Akta Kelahiran

Orang tua tidak dapat mengubah informasi, nilai, absensi, jadwal, catatan guru, pesan, maupun galeri.

## Menjalankan
Buka CMD pada folder yang berisi package.json, kemudian:

npm install
npm run dev

Untuk build:

npm run build
