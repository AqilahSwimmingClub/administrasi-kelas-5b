# Pembaruan Foto Siswa

- Guru dapat mengunggah, mengganti, dan menghapus foto dari menu Data Siswa.
- Foto otomatis tampil di portal orang tua.
- Orang tua dapat mengganti foto anak dari profil portal orang tua.
- Perubahan orang tua otomatis tampil pada halaman guru.
- Foto dikompresi otomatis sebelum disimpan agar sinkronisasi lebih ringan.
- Bila Supabase aktif, foto ikut tersinkron antara HP dan website bersama data kelas.
