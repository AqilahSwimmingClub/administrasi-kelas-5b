# Pembaruan Modul e-Rapor Kelas 5B

Pembaruan ini menambahkan menu tanpa menghapus fitur lama:

1. Tujuan Pembelajaran
   - Pilih mata pelajaran dan semester.
   - Tambah, edit, dan hapus TP.
   - TP menjadi dasar deskripsi nilai rapor.

2. Input Nilai Rapor
   - Input nilai manual per siswa.
   - Pilih TP tercapai optimal dan TP yang perlu peningkatan.
   - Impor file Excel.
   - Download format Excel berdasarkan data siswa Kelas 5B.
   - Pembaca impor kompatibel dengan format contoh e-Rapor yang diberikan dan format sederhana aplikasi.

3. Nilai Tersimpan
   - Cek nilai rapor.
   - Cek deskripsi nilai rapor otomatis.
   - Hapus satu nilai atau semua nilai pada mapel/semester terpilih.

4. Cek Penilaian
   - Status jumlah nilai dan deskripsi setiap mata pelajaran.
   - Capaian nilai rapor per siswa.
   - Grafik perkembangan nilai per semester.

Data baru yang ikut disimpan dan disinkronkan:
- learningObjectives
- reportGrades

Contoh file impor disertakan dalam folder `contoh-import`.
