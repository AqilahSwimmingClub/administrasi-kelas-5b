# Pembaruan Pemilih Tanggal

- Semua formulir tanggal menggunakan tiga pilihan terpisah: Tanggal, Bulan, dan Tahun.
- Diterapkan pada tanggal lahir siswa, jadwal/agenda, dan pengumuman.
- Tampilan tanggal menggunakan format Indonesia, misalnya `23 Januari 1991`.
- Absensi tidak menggunakan pemilih tanggal. Tanggal absensi terisi otomatis sesuai tanggal perangkat saat halaman dibuka.
- Nilai tanggal tetap disimpan dalam format `YYYY-MM-DD` agar kompatibel dengan sinkronisasi dan pengurutan data.
