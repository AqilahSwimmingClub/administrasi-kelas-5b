# Perbaikan Background dan Form Login

Pembaruan ini hanya memperbaiki tampilan tanpa mengubah data atau fungsi utama aplikasi:

- Susunan kolom username/NISN dan kata sandi dikunci horizontal agar ikon, teks, dan tombol mata tidak bertumpuk.
- Background `login-master-bg.jpg` dipertahankan pada halaman login.
- Dashboard/portal guru menggunakan background yang sama melalui `.app-shell`.
- Portal orang tua menggunakan background yang sama melalui `.parent-only`.
- Lapisan gelap transparan dipakai pada dashboard agar teks dan tabel tetap jelas.

Setelah upload ke GitHub dan Vercel berstatus Ready, lakukan hard refresh `Ctrl + Shift + R` atau buka mode Incognito.
