# Perbaikan Login

Pembaruan ini hanya memperbaiki tata letak box login dan portal orang tua.

- Background login tetap memakai `public/login-master-bg.jpg`.
- Ikon, username/NISN, password, dan tombol mata tersusun satu baris.
- Box login diperkecil agar tulisan WELCOME TO CLASS 5B tetap terlihat.
- Portal orang tua memakai background yang sama dengan lapisan gelap agar teks jelas.
