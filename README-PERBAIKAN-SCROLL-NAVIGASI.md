# Perbaikan Scroll Navigasi

Navigasi samping sekarang dapat digulir ke atas dan ke bawah pada desktop maupun HP.

Perubahan hanya pada tampilan sidebar:
- daftar menu memiliki area scroll sendiri;
- mendukung sentuhan pada HP;
- tombol Keluar dan kredit tetap berada di bagian bawah;
- tidak ada menu atau fitur aplikasi yang dihapus.
