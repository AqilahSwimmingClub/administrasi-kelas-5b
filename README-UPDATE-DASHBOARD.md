# Pembaruan Dashboard Army Gold

Pembaruan ini menerapkan dashboard guru bergaya army–gold yang konsisten dengan halaman login:

- Sidebar gelap dengan ikon emas.
- Header sekolah dan profil guru.
- Hero sambutan Fahmi Djawas, S.Pd.
- Enam kartu statistik.
- Grafik kehadiran.
- Distribusi nilai.
- Daftar siswa terbaik.
- Aktivitas, pengumuman, dan agenda.
- Tetap mempertahankan Data Siswa, Absensi, Penilaian, Portal Orang Tua, Pengaturan, Supabase, dan cetak buku nilai.
